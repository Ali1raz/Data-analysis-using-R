## Step 1 ŌĆö Import & Preprocess

**Syntax model:** `data()` ŌåÆ `str()` ŌåÆ `summary()` ŌåÆ clean ŌåÆ factor

```r
data(iris)

str(iris)
summary(iris)
head(iris)

# Check missing values
sum(is.na(iris))
colSums(is.na(iris))

# Ensure target is factor
iris$Species <- as.factor(iris$Species)

# Scale numeric features (important for Neural Network)
iris_scaled <- iris
iris_scaled[, 1:4] <- scale(iris[, 1:4])
```

**How to remember:**
- `str()` = types, `summary()` = stats ŌĆö always both
- `as.factor()` = tells R this is a category not a number
- `scale()` = normalize to mean 0, sd 1 ŌĆö required for neural nets, harmless for trees

## Step 2 ŌĆö Train/Test Split

**Syntax model:** `set.seed()` ŌåÆ `sample()` ŌåÆ `[idx,]` / `[-idx,]`

```r
set.seed(42)
n <- nrow(iris_scaled)
train_idx <- sample(1:n, size = 0.7 * n)

train <- iris_scaled[train_idx, ]
test  <- iris_scaled[-train_idx, ]

nrow(train)  # 105
nrow(test)   # 45
```

**How to remember:** split once, reuse `train` and `test` for all three models.

---

## Step 3 ŌĆö Decision Tree

**Syntax model:** `rpart(Y ~ ., data)` ŌåÆ `predict(, type="class")` ŌåÆ `confusionMatrix()`

```r
library(rpart)
library(caret)

# Train
dt_model <- rpart(Species ~ ., data = train, method = "class")

# Predict
dt_pred <- predict(dt_model, newdata = test, type = "class")

# Evaluate
dt_cm <- confusionMatrix(dt_pred, test$Species)
dt_cm
```

**Visualize the tree:**
```r
library(rpart.plot)
rpart.plot(dt_model, type = 4, extra = 101)
```

**How to remember:**
- `Y ~ .` = predict Y using all other columns (`.` = everything)
- `method = "class"` = classification not regression
- `type = "class"` in predict = give me the class label not probabilities

**What a decision tree does:**
> Splits data by asking yes/no questions: "Is petal length < 2.5? ŌåÆ setosa. Else ŌåÆ ask next question." Like a flowchart built from data.

---

## Step 4 ŌĆö Random Forest

**Syntax model:** `randomForest(Y ~ ., data, ntree)` ŌåÆ `predict()` ŌåÆ `confusionMatrix()`

```r
library(randomForest)

# Train
rf_model <- randomForest(Species ~ ., data = train, ntree = 100, importance = TRUE)

# Predict
rf_pred <- predict(rf_model, newdata = test)

# Evaluate
rf_cm <- confusionMatrix(rf_pred, test$Species)
rf_cm

# Feature importance
importance(rf_model)
varImpPlot(rf_model)
```

**How to remember:**
- `ntree = 100` = build 100 trees, take majority vote
- `importance = TRUE` = track which features matter most
- Random Forest = many decision trees voting together ŌåÆ more accurate, less overfitting

**Decision Tree vs Random Forest:**

| | Decision Tree | Random Forest |
|---|---|---|
| How many trees | 1 | 100+ |
| Overfitting | prone | resistant |
| Interpretable | yes (plot it) | harder |
| Accuracy | lower | higher |

## Step 5 ŌĆö Neural Network

**Syntax model:** `nnet(Y ~ ., data, size, maxit)` ŌåÆ `predict(, type="class")`

```r
library(nnet)

# Train
nn_model <- nnet(Species ~ ., data = train, size = 5, maxit = 200)
# size = number of neurons in hidden layer
# maxit = max training iterations

# Predict
nn_pred <- predict(nn_model, newdata = test, type = "class")
nn_pred <- factor(nn_pred, levels = levels(test$Species))

# Evaluate
nn_cm <- confusionMatrix(nn_pred, test$Species)
nn_cm
```

**How to remember:**
- `size` = neurons in the hidden layer ŌĆö start with 5, tune up/down
- `maxit` = how long it trains ŌĆö more = better fit but slower
- `type = "class"` = return class labels not probabilities
- Neural net needs scaled data ŌĆö that's why we did `scale()` in Step 1

**What a neural network does:**
> Passes data through layers of math nodes. Each node learns a weight. Together they learn non-linear patterns trees can't see.

## Step 6 ŌĆö Evaluation Metrics

### Extract from confusionMatrix

```r
# confusionMatrix() gives everything ŌĆö extract like this:

# Accuracy
dt_cm$overall["Accuracy"]
rf_cm$overall["Accuracy"]
nn_cm$overall["Accuracy"]

# Per-class: Precision, Recall, F1
dt_cm$byClass[, "Precision"]
dt_cm$byClass[, "Recall"]
dt_cm$byClass[, "F1"]
```

### What each metric means

| Metric | Formula | Meaning |
|---|---|---|
| **Accuracy** | correct / total | overall % right |
| **Precision** | TP / (TP + FP) | of predicted positives, how many are actually positive |
| **Recall** | TP / (TP + FN) | of actual positives, how many did we catch |
| **F1-Score** | 2 ├Ś (P ├Ś R) / (P + R) | balance between precision and recall |

**Scenario ŌĆö why F1 matters:**
> Spam filter: 95% accuracy sounds great. But if it marks every email as not-spam, it's 100% recall for non-spam, 0% for spam. F1 catches this ŌĆö accuracy alone lies.

### Confusion Matrix layout

```
                Predicted
                setosa  versicolor  virginica
Actual setosa  [  TN  ] [   FP   ] [   FP  ]
       versic  [  FN  ] [   TP   ] [   FP  ]
       virgin  [  FN  ] [   FP   ] [   TP  ]

Diagonal = correct predictions
Off-diagonal = mistakes
```

## Step 7 ŌĆö Full Comparison

```r
# Accuracy comparison
cat("Decision Tree Accuracy :", round(dt_cm$overall["Accuracy"], 3), "\n")
cat("Random Forest Accuracy :", round(rf_cm$overall["Accuracy"], 3), "\n")
cat("Neural Network Accuracy:", round(nn_cm$overall["Accuracy"], 3), "\n")

# F1 per model (averaged across classes)
mean_f1 <- function(cm) mean(cm$byClass[, "F1"], na.rm = TRUE)

cat("DT  F1:", round(mean_f1(dt_cm), 3), "\n")
cat("RF  F1:", round(mean_f1(rf_cm), 3), "\n")
cat("NN  F1:", round(mean_f1(nn_cm), 3), "\n")
```

**Typical results on iris:**

| Model | Accuracy | F1 |
|---|---|---|
| Decision Tree | ~0.93 | ~0.93 |
| Random Forest | ~0.96 | ~0.96 |
| Neural Network | ~0.95 | ~0.95 |

## Step 8 ŌĆö Visualizations

```r
library(ggplot2)

# --- Decision Tree plot ---
rpart.plot(dt_model, type = 4, extra = 101,
           main = "Decision Tree - Iris")

# --- Random Forest: feature importance ---
varImpPlot(rf_model, main = "Random Forest - Feature Importance")

# --- Confusion matrix heatmap for any model ---
plot_cm <- function(cm, title) {
  cm_table <- as.data.frame(cm$table)
  ggplot(cm_table, aes(x = Prediction, y = Reference, fill = Freq)) +
    geom_tile() +
    geom_text(aes(label = Freq), size = 5) +
    scale_fill_gradient(low = "white", high = "steelblue") +
    labs(title = title) +
    theme_minimal()
}

plot_cm(dt_cm, "Decision Tree Confusion Matrix")
plot_cm(rf_cm, "Random Forest Confusion Matrix")
plot_cm(nn_cm, "Neural Network Confusion Matrix")

# --- Accuracy bar comparison ---
acc_df <- data.frame(
  Model    = c("Decision Tree", "Random Forest", "Neural Network"),
  Accuracy = c(dt_cm$overall["Accuracy"],
               rf_cm$overall["Accuracy"],
               nn_cm$overall["Accuracy"])
)

ggplot(acc_df, aes(x = Model, y = Accuracy, fill = Model)) +
  geom_bar(stat = "identity", width = 0.5) +
  ylim(0, 1) +
  geom_text(aes(label = round(Accuracy, 3)), vjust = -0.5) +
  labs(title = "Model Accuracy Comparison") +
  theme_minimal()
```
