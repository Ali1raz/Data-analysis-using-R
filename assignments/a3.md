## Full Regression Program in R — With Syntax Models & Memory Tricks

---

## Mental Model First

```
Data → Clean → Split → Train Models → Evaluate → Visualize → Compare
```

Every step feeds the next. Never skip order.

---

## Step 1 — Import & Explore

**Syntax model:** `data(name)` → `str()` → `summary()`

```r
data(mtcars)

str(mtcars)       # column types — always first
summary(mtcars)   # min, max, mean per column
head(mtcars)      # first 6 rows
nrow(mtcars)      # how many rows total
```

**How to remember:** `str` = structure, `summary` = statistics — do both, always.

---

## Step 2 — Preprocessing & Missing Values

**Syntax model:** `is.na()` → fix → `as.factor()`

```r
# Check
sum(is.na(mtcars))
colSums(is.na(mtcars))

# Fix option 1 — remove rows
clean <- na.omit(mtcars)

# Fix option 2 — replace with mean
mtcars$mpg[is.na(mtcars$mpg)] <- mean(mtcars$mpg, na.rm = TRUE)

# Convert binary target to factor for logistic
mtcars$am <- as.factor(mtcars$am)
```

**How to remember:**
- `is.na` = is it NA?
- `na.omit` = omit NAs
- `as.factor` = treat as category not number

---

## Step 3 — Train/Test Split

**Syntax model:** `set.seed()` → `sample()` → `df[idx,]` / `df[-idx,]`

```r
set.seed(42)                                  # freeze randomness
n <- nrow(mtcars)                             # count rows = 32
train_idx <- sample(1:n, size = 0.7 * n)     # pick 70% row numbers randomly

train <- mtcars[train_idx, ]                  # keep those rows
test  <- mtcars[-train_idx, ]                 # keep everything else
```

**How to remember:**
- `set.seed` = same result every time
- `sample(1:n, size)` = randomly pick row numbers
- `[idx,]` = keep rows | `[-idx,]` = remove rows → what's left = test

---

## Step 4 — Simple Linear Regression

**Syntax model:** `lm(Y ~ X, data)` → `summary()`

```r
slr_model <- lm(mpg ~ wt, data = train)
summary(slr_model)
```

**Reading output:**
```
Coefficients:
             Estimate
(Intercept)   37.28     ← mpg when wt = 0
wt            -5.34     ← each +1 weight = -5.34 mpg

R-squared: 0.75        ← weight explains 75% of mpg variation
p-value < 0.05         ← predictor is significant
```

**How to remember:** `Y ~ X` = "Y is predicted by X" — the `~` means "depends on"

---

## Step 5 — Multiple Linear Regression

**Syntax model:** `lm(Y ~ X1 + X2 + X3, data)` → `vif()`

```r
mlr_model <- lm(mpg ~ wt + hp + cyl, data = train)
summary(mlr_model)

# Check multicollinearity
library(car)
vif(mlr_model)    # values > 5 = problem, predictors too similar
```

**How to remember:**
- Same as SLR but add more X's with `+`
- VIF = "are my predictors copying each other?" — keep it low

---

## Step 6 — Logistic Regression

**Syntax model:** `glm(Y ~ X1 + X2, data, family = binomial)`

```r
log_model <- glm(am ~ wt + hp, data = train, family = binomial)
summary(log_model)

# Get predicted probabilities
log_probs <- predict(log_model, newdata = test, type = "response")

# Convert to 0/1 class
log_pred <- ifelse(log_probs > 0.5, 1, 0)
log_pred <- factor(log_pred, levels = c(0, 1))
```

**How to remember:**
- `glm` not `lm` — g = generalized
- `family = binomial` = tells R this is 0/1 classification
- `type = "response"` = give me probabilities not log-odds
- `ifelse(prob > 0.5, 1, 0)` = probability → class

---

## Step 7 — Evaluation

### R-Squared (Linear Models)

**Syntax model:** `summary(model)$r.squared` or manual formula

```r
# Training R²
summary(slr_model)$r.squared
summary(mlr_model)$r.squared

# Test R² — manual
slr_pred <- predict(slr_model, newdata = test)

ss_res <- sum((test$mpg - slr_pred)^2)          # error of model
ss_tot <- sum((test$mpg - mean(test$mpg))^2)    # error of just using mean
r2 <- 1 - (ss_res / ss_tot)
r2
```

**How to remember:**
- `ss_res` = sum of squared residuals = how wrong your model is
- `ss_tot` = total variance = how wrong a flat average would be
- R² = 1 - (your error / baseline error) → closer to 1 = better

---

### Confusion Matrix & Accuracy (Logistic)

**Syntax model:** `confusionMatrix(predicted, actual)`

```r
library(caret)
confusionMatrix(log_pred, test$am)

# Manual accuracy
mean(log_pred == test$am)
```

**Reading confusion matrix:**
```
              Predicted 0   Predicted 1
Actual 0    [ TN ]        [ FP ]
Actual 1    [ FN ]        [ TP ]

Accuracy = (TN + TP) / total rows
```

**How to remember:**
- Diagonal = correct predictions
- Off-diagonal = mistakes
- Accuracy = right answers / all answers

---

### Residual Analysis (Linear)

**Syntax model:** `residuals(model)` → `shapiro.test()` → `plot(model)`

```r
res <- residuals(slr_model)

mean(res)             # should be ~0
shapiro.test(res)     # p > 0.05 = residuals are normal = good

plot(slr_model)       # 4 diagnostic plots automatically
```

**4 plots from `plot(model)`:**

| Plot | What to look for |
|---|---|
| Residuals vs Fitted | random scatter = good, pattern = bad |
| Q-Q Plot | points on diagonal line = normal residuals |
| Scale-Location | flat line = equal variance |
| Residuals vs Leverage | no points past Cook's distance line |

## Step 8 — Visualizations

```r
library(ggplot2)

# SLR — scatter + regression line
ggplot(train, aes(x = wt, y = mpg)) +
  geom_point(color = "steelblue") +
  geom_smooth(method = "lm", se = TRUE, color = "red") +
  labs(title = "SLR: MPG ~ Weight")

# MLR — actual vs predicted
mlr_pred <- predict(mlr_model, newdata = test)
ggplot(data.frame(actual = test$mpg, predicted = mlr_pred),
       aes(x = actual, y = predicted)) +
  geom_point(color = "purple") +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed") +
  labs(title = "MLR: Actual vs Predicted")

# Logistic — probability curve
ggplot(train, aes(x = wt, y = as.numeric(as.character(am)))) +
  geom_point(color = "darkgreen") +
  geom_smooth(method = "glm",
              method.args = list(family = "binomial"),
              se = FALSE, color = "orange") +
  labs(title = "Logistic: P(Manual) ~ Weight", y = "Probability")

# Residual plot
plot(slr_model, which = 1)   # residuals vs fitted
plot(slr_model, which = 2)   # Q-Q normality
```

**How to remember ggplot structure:**
```
ggplot(data, aes(x, y))   ← what and where
  + geom_point()          ← how to draw it
  + labs()                ← labels
```

## Step 9 — Compare All Models

```r
# SLR test R²
slr_pred <- predict(slr_model, newdata = test)
r2_slr <- 1 - sum((test$mpg - slr_pred)^2) / sum((test$mpg - mean(test$mpg))^2)

# MLR test R²
mlr_pred <- predict(mlr_model, newdata = test)
r2_mlr <- 1 - sum((test$mpg - mlr_pred)^2) / sum((test$mpg - mean(test$mpg))^2)

# Logistic accuracy
log_acc <- mean(log_pred == test$am)

cat("SLR  R²:", round(r2_slr, 3), "\n")
cat("MLR  R²:", round(r2_mlr, 3), "\n")
cat("LOG Acc:", round(log_acc, 3), "\n")
```

## Master Cheat Sheet

| Task | Function | Key argument |
|---|---|---|
| Load data | `data()` / `read.csv()` | dataset name |
| Check types | `str()` | — |
| Missing values | `colSums(is.na())` | — |
| Split data | `sample()` | `size = 0.7 * n` |
| SLR | `lm(Y ~ X, data)` | one X |
| MLR | `lm(Y ~ X1+X2, data)` | multiple X |
| Logistic | `glm(..., family=binomial)` | binary Y |
| Predict | `predict(model, newdata)` | `type="response"` for logistic |
| R² | `summary(model)$r.squared` | — |
| Confusion matrix | `confusionMatrix()` | caret package |
| Residuals | `plot(model)` | `which = 1` or `2` |
| Plot | `ggplot() + geom_*()` | aes(x, y) |
